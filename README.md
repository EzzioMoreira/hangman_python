# Hangman python

O jogo da forca em python, jogador escolhe um dos três níveis de deficuldade: 

- `FACIL`: palavras de 3 a 5 caracteres.
- `NORMAL`: palavras de 6 a 10 caracteres.
- `DIFICIL`: palavras com mais de 10 caracteres. 

### Executando jogo

```shell
python ./hangman.py
```
